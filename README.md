# 📝 Vanilla JS To-Do App

Интерактивное и доступное приложение для управления списком задач, написанное на чистом JavaScript.

## ✨ Особенности (Features)
- 🌓 **Кастомный чекбокс**: скрытый дефолтный инпут, управляемый через `<label>` со встроенными SVG-иконками из спрайта.
- 🔄 **Переиспользуемое модальное окно**: одна форма используется как для создания новых задач, так и для редактирования текущих без дублирования элементов в DOM.
- 💾 **Сохранение данных**: задачи автоматически сохраняются в `localStorage` и не пропадают после перезагрузки страницы.
- 🌐 **Интернационализация (i18n)**: полная поддержка нескольких языков (Русский / English) с использованием словаря переводов на базе дата-атрибутов.
- 🟰  **Простота использования**
## 🛠️ Технологии
- HTML5 (Семантическая разметка, SVG-спрайты)
- CSS3 (Flexbox/Grid, кастомные селекторы `:checked + label`)
- JavaScript Vanilla (ES6+, DOM Manipulation, LocalStorage, i18n)

## 🚀 Как запустить проект локально
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Откройте файл `index.html` в любом современном браузере (или запустите через расширение Live Server в VS Code).
